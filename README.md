## 我们在做的事

生活有时候让人喘不过气——太多选择、太多事情、太少被真正听见的时刻。我们在构建真正支持你日常生活的 AI 朋友。一个真正理解你的伙伴，可以帮你理清思绪、感到踏实、更好地面对每一天。

## 人格与记忆

通过人格设计、记忆系统和情感理解，我们创造了一个感觉真实的体验——它会倾听、会回应、会记住你，也知道什么时候该退后一步。

## 开源透明

我们相信 AI 伴侣应该是透明的。OpenPat 完全开源——你可以看到它的每一行代码。
